# Association-Rule-Mining-Heart-Disease-
Homework assignment for the course Data Mining and Machine Learning.
This project was created as part of an academic assignment focused on reproducing scientific research results, understanding association-rule techniques, and evaluating differences between original and reproduced findings.

#### Project Overview
- Loaded and preprocessed the Cleveland Heart Disease dataset
- Encoded features for association rule mining
- Applied Apriori algorithm
- Extracted and analyzed frequent itemsets
- Compared rules for male vs. female subgroups
- Documented differences between reproduced and published results
